# Skill: テスト手続き (Test)

## 概要
成果物の品質を定量的・定性的に検証し、動作を保証する。

## Instructions
1. **テスト項目作成**: 正常系、異常系、境界値を含むテストケースを列挙する。
2. **環境準備**: テストに必要なデータやモックを用意する。
3. **実施**: 自動テストまたは手動検証を行い、ログを記録する。
4. **証跡管理**: 成功結果（スクリーンショット、ログ、テストレポート等）を pmMode に提示する。
