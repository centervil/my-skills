# Skill: リポジトリ・監査 (Auditor)

## 概要
リポジトリの物理的なファイル構成、規約、整合性を包括的に監査し、設計思想（README/AGENTS）との乖離を特定する。

## Instructions
1. **リポジトリ全件監査 (Repository Audit)**:
   - 物理構成と README.md の整合性確認。
   - リンク切れ、不要ファイルの検出。
2. **プロジェクト別監査 (Project Audit)**:
   - ソースコードとテストコードの対応関係の確認。
   - E2E テストの実行と結果の検証。
3. **監査レポート**:
   - 監査結果（OK/NG）のリスト。
   - 検出された問題の詳細と推奨される修正アクション。

## Output Contract
- 監査結果を構造化した Markdown 形式のレポート
- 修正が必要なファイルと箇所、およびその修正提案
